## 安装依赖

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### 启动项目

```bash
npm start
```

### 打包

```bash
npm run build
```

### 检查代码格式

```bash
npm run lint
```

也可以自动修复

```bash
npm run lint:fix
```

### 测试代码

```bash
npm test
```
